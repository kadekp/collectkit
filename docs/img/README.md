# Image assets

Drop screenshots here and uncomment the matching `<!-- ![]() -->` block
in the top-level `README.md`.

| Filename                  | What it should show                                                                 |
|---------------------------|-------------------------------------------------------------------------------------|
| `whatsapp-screenshot.png` | A sanitized real WhatsApp conversation with the bot. Goes under "See it in action". |
| `langsmith-trace.png`     | A LangSmith trace of one agent turn — prompt, tool calls, response. Optional.       |

Tips for the WhatsApp screenshot:
- Crop tightly to the conversation bubbles; no contact bar.
- Mask or rename phone numbers and full names.
- 2–3 turns is enough — show one tool call landing (e.g. PTP recorded).
- ~600–800px wide renders cleanly on GitHub.
